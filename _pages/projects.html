Operations Research Project

Multi-Depot Vehicle Routing Problem with Heterogeneous Fleet

Optimization framework for last-mile delivery with EV/GV mixed fleet across 3 depots

Abstract: This project presents a computational framework for solving the Multi-Depot Heterogeneous Fleet Vehicle Routing Problem (MDHFVRP) in last-mile delivery logistics. The system optimizes delivery routes across K = 3 distribution centers serving N = 69 geographically dispersed customers using a mixed fleet of Electric Vehicles (EVs) and Gasoline Vehicles (GVs).

Download Technical Report View on GitHub

Mathematical Formulation: The objective is to minimize total operational cost across all depots, comprising distance-based vehicle operating costs and time-based labor costs. The problem is formulated as a Mixed Integer Linear Program (MILP) with constraints for customer visits, flow conservation, vehicle range, and working time limits.

Methodology: The solution approach employs a decomposition strategy where each depot's subproblem is optimized independently using Google OR-Tools. The framework uses PATH_CHEAPEST_ARC heuristic for initial solution generation and Guided Local Search (GLS) metaheuristic for solution improvement.

Master's Thesis

A Machine Learning Approach for Evaluating Climate Sensitivity of Infectious Diseases in Refugee Communities

ML framework with SHAP interpretability for climate-disease prediction using VAE data augmentation

Abstract: Globalization, climate change, and increased population mobility have intensified the challenges of infectious disease outbreaks for global public health. Refugee communities are particularly vulnerable due to overcrowding, inadequate sanitation, and delayed healthcare access. This study develops a robust, interpretable machine learning framework to identify climatic factors contributing to infectious disease incidence.

Methodology: The framework integrates meta-heuristics feature selection (PSO, Genetic Algorithm) to select relevant variables. Time Variational Auto-Encoder (Time-VAE) addresses data scarcity. SHAP values provide feature importance explanations with enhanced interpretability.

Key Findings: VAE significantly improves predictive performance against limited sample size. BART best captures complex nonlinear climate-disease relationships. The framework provides actionable insights for public health planning in resource-constrained settings.

Undergraduate Thesis

Industry 4.0 Readiness Assessment Framework and Barrier Analysis

AHP-based framework for evaluating digital transformation readiness across 7 dimensions

Overview: This thesis presents a comprehensive framework for evaluating organizational readiness for Industry 4.0 digital transformation. The research identifies and prioritizes implementation barriers using hierarchical weighting analysis across technology, organization, human resources, and regulatory dimensions.

Methodology: Analytic Hierarchy Process (AHP) with weighted scoring evaluates organizational readiness. A comprehensive barrier analysis identifies 17 sub-barriers categorized under four major barrier types with normalized global weights.
