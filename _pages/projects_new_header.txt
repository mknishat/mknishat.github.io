    <h2 style="font-size: 18px; color: #333; margin: 15px 0 10px 0;">Multi-Depot Vehicle Routing Problem with Heterogeneous Fleet</h2>

    <p style="font-size: 15px; text-align: justify; color: #555; line-height: 1.6; margin: 15px 0;">
      <strong>Abstract:</strong> This project presents a computational framework for solving the <strong>Multi-Depot Heterogeneous Fleet Vehicle Routing Problem (MDHFVRP)</strong> in last-mile delivery logistics. The system optimizes delivery routes across K = 3 distribution centers serving N = 69 geographically dispersed customers using a mixed fleet of Electric Vehicles (EVs) and Gasoline Vehicles (GVs).
    </p>

    <p style="font-size: 15px; text-align: justify; color: #555; line-height: 1.6; margin: 15px 0;">
      <strong>Mathematical Formulation:</strong> The objective is to minimize total operational cost across all depots, comprising distance-based vehicle operating costs and time-based labor costs. The problem is formulated as a Mixed Integer Linear Program (MILP) with constraints for customer visits, flow conservation, vehicle range, and working time limits.
    </p>

    <div style="margin: 20px 0; text-align: center;">

    <p style="font-size: 15px; text-align: justify; color: #555; line-height: 1.6; margin: 15px 0;">
      <strong>Methodology:</strong> The solution approach employs a decomposition strategy where each depot's subproblem is optimized independently using Google OR-Tools. The framework uses PATH_CHEAPEST_ARC heuristic for initial solution generation and Guided Local Search (GLS) metaheuristic for solution improvement.
    </p>

      <strong>Key Findings:</strong> (1) Fleet electrification reduces costs by <strong>9.1%</strong> while maintaining service levels; (2) K-means clustering effectively partitions customers for multi-depot operations; (3) Average vehicle utilization of <strong>51.3%</strong> indicates potential for fleet size optimization; (4) Guided Local Search achieves near-optimal solutions within 5 seconds per depot.
    </p>
